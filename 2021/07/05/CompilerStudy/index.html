CompilerStudy

Author: shuisong Date: 七月 5, 2021  22:11:51 Category: compiler

 重新学习一下编译原理吧，之前学习的时候对于部分内容理解并没有到位，第二次学习的时候得下的功夫多很多啊，这个就好比煮开水，第一次就烧开的话第二次只需要放一点儿小火就行了，那要是第一次没到位，后边儿又得添加很多的柴火了。重新添加柴火表示很恼火。

  • 课程链接：http://web.stanford.edu/class/cs143/
  • 学习步骤：PPT->book->assignment->project
  • 时间规划：3h*30

Cool Compiler

全称Classroom Object Oriented Language，用于教学的一个编译器，麻雀虽小五脏俱全。

数据类型

  • object：类、初始化、继承
  • int：整形
  • bool：布尔变量
  • string：字符串

    表达式

    每个表达式都含有类型和值
  • Loops：while E loop E pool
  • Conditionals： if E then E else E fi
  • Case statement： case E of x : Type Þ E; … esac
  • Arithmetic, logical operations
  • Assignment： x ¬ E
  • Primitive I/O： out_string(s), in_string(), …
    未实现的特点
  • arrays, floating point operations, exceptions, …

    内存管理

  • 分配：每次new的时候进行内存的分配
  • GC：自动回收，当内存unreachable的时候
